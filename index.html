Download Feature

見つけたら、保存する。

対応サイトの動画・音声・画像を直接保存。手順は4つだけ。

  1. DLボタンをタップ多くの場合、先に動画を再生しておくのがコツ。
  2. ファイルを選択Unknown SD/HD・HLS・GIFなどから選ぶ。
  3. ファイル名と種類を確認Execute / Cancel を選択。
  4. ダウンロード実行失敗してもDL回数は減らない。安心して挑め。
Statusサイト例
✅ 対応(HLS/MP4)一般的なHLS/MP4ストリーム配信サイト
🔗 リダイレクト(外部DL)YouTube
⚠️ 音声なしX(Twitter)
❌ 非対応(DRM)Netflix / Amazon Prime Video など

400 DLシステムエラー / 404 リンク切れ / 500 サイト側ブロック

※ 対応サイトは予告なく変更される場合があります。ダウンロードは合法の範囲で、自己責任で。

  2. 下のURLをコピー

  2. iPhoneにAltStoreをインストール

  4. AltStore経由でIPAをインストール

アップデートは手動。新しい.ipaを毎回インストールしてください。

FAQ

だいたい全部、答えておく。

なんでApp Storeにないの?

多くの機能がAppleのガイドラインに抵触するため、App Storeでの配布は不可能です。だからAltStore。

中華アプリでしょ?データ抜いてるんでしょ?スパイ!

開発者は日本在住の個人です。ソースは非公開ですが、データは一切収集していません。仮に集めたとしても、ストレージ代で破産します。

無料で使える?

使えます。Freeプランあり。ただしDL回数に制限(10回/日)があるので、ヘビーユーザーは有料プランをどうぞ。

違法じゃないの?

このソフトウェアは単なるブラウザであり、違法行為を助長するものではありません。合法の範囲で、自己責任でご利用ください。

Android版は?

現在iOSのみ。将来的には検討するかも。(正直に言うとAndroidユーザーは課金しないし、ModApkで即クラックされるので、たぶん出ません。)

Windows / Mac版は?

いつか対応できたらいいですね。でもPCで「アレ」を見る人、そんなに多くないので……。

逆コンパイルして課金機能を無料で使っていい?

ふざけるな。それは開発者を殺す行為です。やめてください。

返金は?

Patreonの返金ポリシーをご確認ください。

Patreonとの連携方法は?

アプリを開いて Others/Billing → Login をタップ。ティアが反映されない場合はティアを確認してRefresh。それでもダメなら開発者まで連絡を。

データを移動したい

FilesアプリからOnMyiPhone/YorunoBrowserにあります。bookmarkもhistoryも消しても復活します。
